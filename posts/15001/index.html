小米12的优化

博主在2023年11月25号在海鲜市场购买了一台小米12，之前是用小米Mix2s的，然后买了一台红米Note11TPro，这台手机被商家坑惨了，闪光灯有问题，屏幕有问题，还有一些奇怪的毛病，完完全全就是一台炸弹机
在2023年11月23号，我把Switch卖了之后就换了小米12了，当时没有看过任何的测评，感觉小米12，曲面屏，小屏，对称扬声器，骁龙8，应该很牛逼吧
曲面屏的手机一直没有体验过，而小屏一直是我所追求的，我自己觉得这俩搭配在一块可以说是绝配
小米12的扬声器音质听感舒适，加上模块加持，有着优秀的音效体验，然而，当玩游戏时，手持姿势可能会无意中挡住扬声器(沉默一下…)
接下来就是火龙8，刷个视频什么的很容易飚到四十多度，如果删除了温控，玩游戏甚至可以到达五十多度，现在知道火龙的威力了
下面是我自己优化的过程，因为我对温度不敏感，只要能流畅就行了

潘多拉内核

潘多拉内核的作用可以降低一些功耗什么的，效果因人而异，下面是刷入教程，潘多拉内核很多人推荐0101版本，下面只提供0101的版本，最新版请去作者频道获取
刷写工具
下载：123云盘
内核下载
0101版本下载: 123云盘
安装好工具和下载好内核后，需要给EX Kernel Manager和Horizon Kernel Flasher这两个软件root权限，首先备份内核，做好备份以防变砖
打开EX Kernel Manager这个软件，点击左上角的菜单按钮

然后选择刷写中心

接着内核备份就行了



做好上面的工作内核就备份好了，内核默认备份是在/Documents/kernel_backups/目录里面，可以拷贝到电脑，以防万一
下面是刷入潘多拉内核的教程
首先打开Horizon Kernel Flasher，选择潘多拉内核文件，接着等待刷入即可，过程会提示重启设备，确认即可
到这里潘多拉内核就刷写好了，如果需要刷回原来的内核的话，打开root管理器，如果你是Kernelsu就打开Kernelsu，是Magisk就打开Magisk，找到Pandora kernel附加模块选择移除

接着打开EX Kernel Manager，点击左上角的菜单按钮，选择刷写中心，然后点击刷入boot镜像….，上次备份的内核文件在/Documents/kernel_backups/，选择上次备份的文件即可，接着点击刷写，过程中会提示重启设备选择重启就可以刷回原来的内核啦




这里我备份了两个内核，如果你跟我一样备份了多个内核，请辨别哪个是手机原来的内核

GPU降压

降压是什么这些问题，可以自行搜索，这里只提供降压作业
降压我们使用的软件是KonaBess
下载链接: Github
下载链接2: 123云盘
进入之后，会要求选择自己的芯片组，点击提示是芯片选项即可，没有提示选项的话，我们可以从头到尾依次慢慢选择自己的芯片

选择完成后，一定要备份旧镜像！！！
备份的镜像默认在存储目录根部，自己保存好，以防万一

这次降压作业使用的是酷安大佬英雄的人民拯救世界
原贴链接: 酷安
之前的步骤做好了后依次点击导入导出-从剪切板导入
下面有两个大佬提供的降压作业，一个是省电版的，一个是完整版超频的，打游戏的话可以选择完整版超频的，大佬推荐使用省电版的

导入完成后点击打包并刷入镜像即可，记得给KonaBess这个软件Root权限

如何确认是否导入成功？
我们可以在KonaBess点击编辑GPU频率表-芯片名称，省电版的频率只有两个，然后进入最新版的Scene，旧版本我没试过能不能行，在功能里面找到CPU控制，往下翻找到GPU，里面的频率跟降压软件的频率对应即可
